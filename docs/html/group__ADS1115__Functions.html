◆ ads1115_continuous_conversion_read()

ads1115_error_t ads1115_continuous_conversion_read ( ads1115_handle_t * handle,
int16_t * adc_raw,
float * voltage )

Reads the latest conversion result in continuous mode.

The voltage is calculated using: $$V_{out} = \text{adc_raw} \times \frac{\text{FSR}}{2^{15}}$$

  • Parameters
    handlePointer to the device handle structure.
    [out]adc_rawRaw 16-bit signed integer output.
    [out]voltageCalculated voltage in Volts.
    Returns
    ads1115_error_t result.
